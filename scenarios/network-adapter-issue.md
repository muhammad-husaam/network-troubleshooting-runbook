## Network Adapter Not Working

---

## 🛠 Problem
The network adapter shows disabled or missing, preventing connectivity.

---

## 🔍 Diagnosis Steps

1. Check adapter status in Control Panel or Settings  
2. Verify adapter in Device Manager  
3. Restart network adapter  
4. Check driver status and update if needed

---

## ✅ Solution

- Enabled network adapter  
- Reinstalled or updated driver  
- Restarted system and verified connectivity
